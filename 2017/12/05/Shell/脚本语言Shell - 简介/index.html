脚本语言shell -简介

发表于 2017-12-05 | 分类于 Shell

####什么是Shell
Shell 脚本（shell script），是一种为 shell 编写的脚本程序。例如:Android开发：动态库编译（.so文件）, iOS开发：静态库（.a文件）都可以使用shell来编译

####Shell环境
Mac环境支持Shell开发
Shell中类非常多,Bash所有的操作系统默认脚本语言（免费、易用）
种类：bash、c Shell、K Shell等等…

####编译第一个shell脚本

  • #! 是一个约定的标记，它告诉系统这个脚本需要什么解释器来执行，即使用哪一种 Shell。

####修改文件权限

  • 文件权限分为4个部分 (类比QQ群)

    • 文件类型: (d:目录 -:文件)
    • 属主权限 (群主)
    • 属组权限 (管理员)
    • 其他用户权限 (其他成员)

  • 查看文件的权限

    • r: 可读
      • w: 可写
      • x: 可执行
      • -:没有该权限就用
  • 修改文件权限

####shell脚本语言注释 (#)

shell脚本中只有单行注释,没有多行注释,单行注释使用 #
